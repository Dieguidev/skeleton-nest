# EcommerceDB API
1. Clonar el repositorio

2. Instalar dependencias
```
npm install
```
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```

4. cambiar las variables de entorno

5. Levantar la base de datos
```
docker-compose up -d
```
6. Levantar en modo desarrollo
```
npm ru8n start:dev
```
